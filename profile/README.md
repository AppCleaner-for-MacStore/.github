## What Files Does AppCleaner Remove?

When you uninstall an app using AppCleaner, it finds and removes all of the following hidden file types:

- **Preference files** — App settings stored in ~/Library/Preferences
- **Cache files** — Stored in ~/Library/Caches
- **Application Support files** — ~/Library/Application Support
- **Log files** — ~/Library/Logs
- **Saved Application State** — ~/Library/Saved Application State
- **Launch Agents** — Background processes the app installed
- **Crash Reports** — ~/Library/Logs/DiagnosticReports
- **Container files** — For sandboxed Mac App Store apps
